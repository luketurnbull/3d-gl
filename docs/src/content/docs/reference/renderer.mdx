Create a new WebGL renderer with the Renderer class.

## Usage

Most simple usage is to pass a `HTMLCanvasElement` to the Renderer class:

```ts
const canvas = document.getElementById("my-canvas-id");
const renderer = new Renderer(canvas);
```

### Renderer Options

#### Background color

The default is transparent, but you can override by passing `backgroundColor` in the options object.

`backgroundColor` takes in an instance of `Color`. See more on the [Color class here](../color).

This is an example of passing a blue background:

```ts
const renderer = new Renderer(canvas, {
   backgroundColor: new Color(1, 0, 0, 1);
})

renderer.render();
```

### Adding meshes

You can add a mesh to the renderer using `addMesh`

`addMesh(mesh: Mesh)`

#### Basic usage

Adding two triangle meshes to the renderer:

```ts
import { Renderer, Mesh } from "@l-studio/3d-gl";

const canvas = document.getElementById(
  "two-triangles-canvas"
) as HTMLCanvasElement;
const renderer = new Renderer(canvas);

// Bottom triangle
renderer.addMesh(
  new Mesh(renderer.gl, new Float32Array([-1, -1, 0, 0, 1, -1]))
);

// Top triangle
renderer.addMesh(new Mesh(renderer.gl, new Float32Array([-1, 1, 0, 0, 1, 1])));

renderer.render();
```
